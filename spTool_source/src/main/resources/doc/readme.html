spTool

Single Particle Tool

For the analysis of single event ICP-MS data.

Hotkeys and key combinations

Control + O Open project

Control + R (Re)process

Control + G Create ('generate')

Control + E Export dialog

Control + I Go to import menu

Alt + I Import dialog

Control + S Save project

Alt + S Save GUI appearance

Control + Shift + S Save method

Control + P Select previous sample

Control + N Select next sample

In most popup windows: Control + Enter executes the main functionality of the popup such as the 'Continue' or 'Run' buttons would. Examples:

  • Execute import
  • Reprocess a sample
Shift + Escape closes most popup windows.

Quick troubleshooting

Installation

spTool is executed from the batch file in the folder you received. If you keep this folder on a network drive (e.g., virtual drives, Google Drive, OneDrive, etc.), you may encounter issues. On university or company systems, the Windows desktop is often mapped to such a network drive. If spTool behaves unexpectedly, try copying the folder to a local path on your machine and run it from there.

Plot Navigation

Go Back to Previous View

Shortcut: Control + Z

Reset graph entirely: Click left mouse and drag up.

Move the Plot

  • Option A: Alt + 1× left mouse click, then move the mouse
  • Option B: 2× left mouse click, then drag the mouse

Zoom

  • Option A: Scroll wheel → zoom both x and y axes
  • Option B: Scroll wheel + Control → zoom x axis
  • Option C: Scroll wheel + Alt → zoom y axis

Arrow Keys

Click on the plot to activate arrow navigation.

  • Arrow keys (Left/Right/Up/Down): Move the plot
  • Control + Left/Right: Zoom in/out for x axis
  • Control + Up/ Down: Zoom in/out for y axis

Dragging zoom

  • Control + Right mouse Zoom x axis by dragging
  • Shift + Left mouse: Zoom y axis by dragging

Transfer values from plots

Double left click on a graph to copy current mouse position. Certain text fields in the user interface accept values when double clicking on the text field while pressing control key.
