# Installation of spTool

## Download spTool

1. Go to https://github.com/melinkmann/spTool3-release/releases/latest.
2. Scroll to the bottom to the section "**Assets**"
3. Download the `.zip` file for you operating system (windows, macOS or Linux are available) and unpack it
4. Execute the **startspTool** file: `startspTool3-Windows.bat`, `startspTool3-Linux` or `startspTool3-macOS`,
   respectively. You may have to confirm that you want to execute the software despite a warning from your
   operating system.

## spTool files on your system

- At first startup, a folder called `spTool3` will be created automatically in your local user location. This folder
  contains the configuration, default settings and **methods**. On a Windows machine, the path is
  `C:\Users\YOURUSERNAME\spTool3`.
