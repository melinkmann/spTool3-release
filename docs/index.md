# spTool documentation ![img](images/spTool_sym_crop10.png)

The single particle tool - `spTool` is a software with graphical user interface
for advanced data analysis in single event ICP-MS including **single particle** and **single cell ICP-MS**.

## Key features

1. generate **in-silico data** of quadrupole and time-of-flight mass spectrometers (spICP-QMS, spICP-TOFMS)

2. **analyze experimental** spICP-MS data including
    - importer for many commercial ICP-MS raw data formats
    - data-dependent thresholding and gating with Gaussian and Poisson statistics
    - multi-event peak filtering
    - cluster and composition analysis
    - calibration for particle number, mass and size quantification

- easy handling of particle populations
- straightforward managing and handling of multiple samples
